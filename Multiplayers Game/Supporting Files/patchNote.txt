                -------------Multiplayers Game Patch Note-------------

V1.2.1 (05/04/2020):
    - Correction d'un bug dans le mode de jeu "race" qui faisait qu'un bouton n'apparaissait pas
    - Correction d'un bug dans le mode de jeu "chateaux" qui faisait que les flèches pouvait ne pas apparaitre pendant un certain temp
    - Ajout d'une icone au jeu
    - Meilleur interface utilisateur pour certaines tailles d'écran
    - Ajout d'un timer pour le mode de jeu race

V1.2.0 (04/04/2020) :
    - Nouveau mode de sélection du nombre de joueurs
    - On peut maintenant sélectionner des couleurs spécifiques
    - Ajout des modes 5 et 6 joueurs. Ils sont pour l'instant seulement disponible sur le jeu race.
    - Le dessin des voitures a été refait

V1.1.0 (23/03/2020):
    - Le mode de jeu "Race" est sorti ! Faites la course avec vos amis et voyez lequel est le plus rapide !

V1.0.1 (05/03/2020):
    - Ajustements du générateur de map pour qu'il prenne moins de temps à créer les maps quatres joueurs.
    - Le menu a été complètement refait pour permettre la création de différent mode de jeu

V1.0 (01/03/2020) :
    - Le jeu Multiplayers Game comporte un mode de jeu : "Chateau".
    - On peut y jouer de 2 à 4 joueurs.
